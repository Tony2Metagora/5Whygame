GitHub Pages = page vitrine. Le bouton « Ouvrir l’app » doit pointer vers l’URL Production affichée sur Vercel (ex. xxx.vercel.app). Repo GitHub → Settings → Secrets and variables → Actions → Variables : PUBLIC_APP_URL = cette URL, puis relancer le workflow « Deploy GitHub Pages ».
